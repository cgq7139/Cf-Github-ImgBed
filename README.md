# Cf-Github-ImgBed

一个基于Cloudflare Workers & Github API的简易图床服务

---

## 简介

Cf-Github-ImgBed 是一个基于 Cloudflare Workers 和 Github API 的简易图床服务。它允许用户通过简单的 HTTP 请求将图片上传到 Github 仓库，并返回图片的 URL。这个服务可以用于在博客、社交媒体或其他需要快速分享图片的场景中。

## 功能

 - 图片上传：
   - 支持拖拽上传
   - 支持粘贴上传
   - 支持URL上传
 - 图片展示：
   - 支持图片预览
   - 支持图片缩放
 - 图片管理：
   - 支持图片删除
